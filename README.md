# FitMe – Aplicación Web de Hábitos Físicos

Aplicación web orientada al seguimiento de hábitos físicos, desarrollada como proyecto personal para fortalecer habilidades en desarrollo web y backend.

## Tecnologías
- Java
- Spring
- HTML, CSS, JavaScript
- MySQL

## Funcionalidades
- Registro de información básica del usuario
- Gestión de datos relacionados con hábitos físicos
- Cálculo y visualización de información relevante

## Estado del proyecto
Proyecto en desarrollo, enfocado en mejora continua y aplicación de buenas prácticas.
